About this tool

Digital Pedagogy Coach Evidence Hub
Version 1.0 · Weston College Group · 2026

Built for Graeme Wright, Digital Pedagogy Coach, Quality & Innovation Team.

This tool runs entirely in your browser. All data is stored in a JSON file you control — nothing is sent anywhere except when you use the Report Builder AI feature (which sends selected evidence entries to the Anthropic API).

Accessibility: Designed to WCAG 2.2 AA standard throughout.
Frameworks: ETF DTPF · DigCompEdu · Jisc BDC · EIF Inclusion · WCAG 2.2
Pilot governance: 3-month review Oct 2026 · 9-month Apr 2027 · 18-month Jul 2027
